  • A markup is a text that has a special meaning.
  • Every element starts and ends with a tag, except the void elements.
    • In an element the first tag is the opening tag and the second one is the closing tag.
    • The terms tags and elements are used exchangeably.
    • '<>' is the opening tag and '</>' is the closing tag
  • We can also use lists in HTML
  • There are two types of lists: ordered and unordered.
    • Ordered lists use the <ol> tag, while unordered lists use the <ul> tag.
    • List items are defined with the <li> tag.
  • Examples of nested lists:

  • Every HTML code has a tree structure, which represents the hierarchy of elements. The root element is the <html> tag, which contains the <head> and <body> sections.
  • Rules for a tree structure:
    1. There is only one root node.
    2. Each node can have barnches that lead to other nodes.
    3. Each node has one parent node, except the root node.
